  • Download and extract the file(s) in a directory called ExtensionInstall/en in your extensions/ folder. If you're a developer and this extension is in a Git repository, then instead you should clone the repository.
  • Add the following code at the bottom of your LocalSettings.php: 
require_once "$IP/extensions/ExtensionInstall/en/ExtensionInstall/en.php";
  • Done! Navigate to "Special:Version" on your wiki to verify that the extension is successfully installed.
